 # CACF Overview

Constitutional AI Compliance Framework operationalizes Articles 14/15/21/21A via:
- Real-time checks (POST /api/v1/cacf/check)
- Bias testing (POST /api/v1/cacf/bias-detect)
- Explainability (POST /api/v1/cacf/explain)
- Audit logging (POST /api/v1/cacf/audit-log)

See: implementation_guide.md and AFL-1/0 integration guides for details.
