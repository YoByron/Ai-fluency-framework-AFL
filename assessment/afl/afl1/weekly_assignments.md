## Week 1-2: Prompt Engineering Fundamentals (10 Assignments)

### Assignment 1.1: Basic Prompt Structure
**Objective:** Understand prompt anatomy

**Task:** Rewrite this vague prompt with proper structure:
- Original: "Tell me about climate change"
- Requirements: Add role, specific task, context, format, constraints

**Success Criteria:**
- Specifies audience (e.g., "Explain to high school students")
- Defines scope (e.g., "Focus on India's climate challenges")
- States format (e.g., "500 words, 3 sections")
- Includes constraints (e.g., "Use simple language, include 2 examples")

**Auto-Grading:** AI Coach scores 0-10 based on completeness

### Assignment 1.2: Zero-Shot vs Few-Shot
**Objective:** Learn when to provide examples

**Task:** Create two prompts for the same task:
1. Zero-shot: "Translate this English text to Hindi"
2. Few-shot: Provide 2 example translations, then ask for translation

**Test Text:** "AI is transforming agriculture through precision farming."

**Success Criteria:**
- Zero-shot prompt is clear and specific
- Few-shot includes 2 quality examples
- Both produce accurate translations
- Learner explains when to use each approach
