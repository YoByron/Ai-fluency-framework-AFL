 # AFL-1 Prompt Lab – Sample Tasks
 
 Provide prompts, iterations, and final outputs. Evaluate each using the checklist (accuracy, sources, bias, privacy, disclosure).
 
 Task A (Education):
 - Generate a lesson outline on “Photosynthesis” for grade 7 with a 5-question quiz; iterate twice to add local examples.
 
 Task B (MSME):
 - Create product page copy and two ad variants for a local bakery; include allergen disclaimers.
 
 Task C (Governance):
 - Draft a response to a citizen RTI-style query with cited sources.
 
 Task D (Healthcare):
 - Write a plain-language patient education handout about hypertension.
 
 Task E (Finance):
 - Summarize market news into a client-safe advisory with risk disclaimers.
