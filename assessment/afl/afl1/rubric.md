# AFL-1 Assessment Rubric – AI Interaction

## Competencies
- Prompting and multi-turn interaction
- Contextualization (domain, language, culture)
- Output evaluation (accuracy, bias, hallucinations)
- CACF-aware everyday use

## Instruments
1. Prompt Lab (5 tasks across domains) – ≥4/5 meet task criteria
2. Evaluation Test – Detect issues in ≥4/5 outputs; correct with citations
3. Compliance Checklist – Apply CACF principles to 3 scenarios

## KPIs
- First-attempt success ≥80% on common tasks
- Measured 15–20% productivity improvement on a personal workflow
