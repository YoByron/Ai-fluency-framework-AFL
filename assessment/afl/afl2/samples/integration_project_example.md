 # AFL-2 Integration Project – Example
 
 ## Process
 Department helpdesk email triage → ticket categorization → response drafting → dashboard.
 
 ## Baseline
 - Avg handling time: 12 min/ticket; 300 tickets/week
 
 ## Toolchain
 - LLM classification + template generator → spreadsheet/API → dashboard
 
 ## CACF Audit
 - Equality: 0.96; Bias: 0.04; Explainability: 0.82; no critical flags
 
 ## Results
 - Handling time reduced to 8 min/ticket (−33%); accuracy maintained; audit logs retained
