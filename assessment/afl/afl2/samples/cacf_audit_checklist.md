 # AFL-2 CACF Audit Checklist (Sample)
 
 - Equality score ≥ 0.95 across demographics (list tested groups)
 - Bias score ≤ 0.05; list flags and mitigations
 - Explainability score ≥ 0.80 for any high-stakes element
 - No critical severity flags; review any medium/high
 - Audit log stored with immutable ID; retention policy applied
