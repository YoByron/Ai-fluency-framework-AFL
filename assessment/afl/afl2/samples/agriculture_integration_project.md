 # AFL-2 Integration Project — Agriculture
 
 ## Process
 Weekly advisories → localized weather/price → farmer summary SMS.
 
 ## Baseline
 - Manual compilation 4 hrs/week; inconsistent quality.
 
 ## Toolchain
 - Mandi prices + weather feeds → LLM summary → voice/SMS delivery.
 
 ## CACF Audit
 - Language fairness focus; equality 0.95; Bias 0.05; Explainability 0.80.
 
 ## Results
 - 2.5 hrs/week (−38%); clearer advisories; increased adoption.
