 # Kalahandi (Odisha) — Rural Challenging Context (AFL-0/1)
 
 ## Context
 - Cohort: 25,000 (AFL-0: 22k, AFL-1: 3k); connectivity/device constraints
 
 ## Problem
 - Limited internet and device access; need offline/voice-first delivery
 
 ## Intervention
 - AFL-0 awareness via facilitators; AFL-1 with voice/SMS support
 
 ## Implementation
 - Heavy on-ground facilitation (1:30)
 - Voice and SMS micro-learning; Odia localization
 - Agriculture-focused examples for relevance
 
 ## Results
 - Completion: 48% (AFL-0), 32% (AFL-1)
 - Productivity: 8% avg for AFL-1 graduates
 - Cost/participant: ₹280 (AFL-0), ₹720 (AFL-1)
 
 ## CACF Compliance
 - Accessibility and language fairness emphasized; audits logged
 
 ## Lessons Learned
 - Offline/voice-first essential; longer timelines required
 
 ## Next Steps
 - Device-lending programs; expand facilitator network
