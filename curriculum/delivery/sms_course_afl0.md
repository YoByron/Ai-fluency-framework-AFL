# AFL-0 SMS-Based Learning Course
## 10-Day AI Awareness via SMS

**Version:** 1.0  
**Last Updated:** November 21, 2025  
**Delivery:** Daily SMS (160 characters max per message)  
**Duration:** 10 days  
**Target:** Low-connectivity areas, feature phone users

---

## Overview

This SMS-based course delivers AFL-0 core concepts via daily text messages, enabling AI awareness for citizens without smartphones or internet access. Each day covers one key topic with interactive quizzes.

**How It Works:**
1. Learners register via SMS: Send "AFL0 [Name] [Language]" to 56070
2. Receive daily SMS at 9 AM for 10 days
3. Reply to quiz questions for interactive learning
4. Receive certificate code upon completion (70% quiz score)

**Languages:** Hindi, English, and 20 regional languages

## Certificate Generation

### Completion Criteria
- Answered all 10 quiz questions
- Scored 7/10 or higher (70%)
- Completed within 15 days of registration

### Certificate Code Format
```
AFL0-SMS-[LANGUAGE]-[YYYYMMDD]-[6-DIGIT-ID]
Example: AFL0-SMS-HI-20251121-123456
```

### Certificate Retrieval
1. Visit afl.gov.in/cert on any device
2. Enter certificate code
3. Download digital certificate (PDF)
4. Share on social media with #AFLIndia

## Technical Implementation

### SMS Gateway Integration
- **Provider:** Govt SMS Gateway / Commercial provider
- **Short Code:** 56070 (dedicated for AFL)
- **Daily Capacity:** 1 million messages
- **Cost:** ₹0.10 per SMS (subsidized for learners)

### Registration Flow
```
User → "AFL0 [Name] [Language]" → 56070
System → Validate format
System → Create learner record
System → Send Welcome SMS
System → Schedule 10 daily messages
```

### Quiz Response Processing
```
User → Reply with A/B/C/D
System → Check answer
System → Update score
System → Send feedback SMS
System → Track progress
```

### Database Schema
```sql
CREATE TABLE sms_learners (
    id SERIAL PRIMARY KEY,
    phone VARCHAR(15) UNIQUE,
    name VARCHAR(100),
    language VARCHAR(10),
    registration_date DATE,
    quiz_scores INT[],
    completion_status BOOLEAN,
    certificate_code VARCHAR(50)
);
```
