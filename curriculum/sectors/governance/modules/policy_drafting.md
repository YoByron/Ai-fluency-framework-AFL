 # Module: Policy Drafting with AI (Governance)
 
 Outcomes: produce policy notes/briefs with citations and bias checks; reduce drafting time.
 
 Activities:
 - Draft a 2-page policy note with cited sources; iterate to add state context.
 - Run bias/sensitivity check on language (religion/caste/gender/region).
 
 Assessment: submit note + citation list + CACF bias check summary (no critical flags).
